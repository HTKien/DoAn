Hướng dẫn cài đặt: 
Bước 1: Chuẩn bị sẵn một máy tính có cài sẵn phần mềm Visual Studio 2019.
Bước 2: Clone Project từ github về máy tính tại đường dẫn sau: http://bit.ly/htk04091998 hoặc Giải nén thư mục mã nguồn
Bước 3: Import Project vào trong Visual Studio.
Bước 4: Kiểm tra các thư viện trong Manage NutGet Packages. Nếu thiếu thư viện nào so với danh sách các thư viện trông file Readme thì cài đặt bổ sung trong phần Manage NuGet Packages.
Bước 5: Rebuid lại toàn bộ ứng dụng và sau đó chạy ứng dụng.
